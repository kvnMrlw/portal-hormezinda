Use `logo/` para armazenar a logo oficial da escola quando o arquivo final estiver disponivel.

Pastas:

- `images/`: imagens gerais do sistema.
- `logo/`: marca oficial da escola.
- `backgrounds/`: fundos visuais.
- `illustrations/`: ilustracoes de suporte.
- `placeholders/`: imagens temporarias.
- `icons/`: icones especificos do produto.
- `fonts/`: fontes locais, caso sejam adotadas.
